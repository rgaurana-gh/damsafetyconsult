Proposed · Advanced Tool

Dam Portfolio Risk
Prioritisation Tool

A tool for state governments and dam owners managing multiple specified dams — enter RRSSD-format risk screening data for each dam, and the tool ranks your portfolio by risk, identifies which failure modes are driving aggregate risk, and generates a Pareto-based priority list for Tier 2 assessment.

Based on NDSA/DRIP RRSSD methodology. The tool does not replicate DHARMA — it provides the portfolio-level interpretation and prioritisation layer that DHARMA does not currently offer.
